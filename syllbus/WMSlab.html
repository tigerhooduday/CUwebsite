Syllabus of WMS LAB

List of Experiments



UNIT-I

  1. Open any website on computer system and identify http packet on monitoring tool like Wireshark.

  2. Design a method to simulate the HTML injections and cross-site scripting (XSS) to exploit the attackers

  3. Implementation of Cross site request forgery (XSRF) attack.

  4. Implementation of Design methods to break authentication schemes (SQL Injection attack).



UNIT-II

  1. Write a program to generate message digest for the given message using the SHA/MD5 algorithm and verify the integrity of message.

  2. Perform Penetration testing on a web application to gather information about the system (Foot Printing).

  3. Implementation of Session hijacking attack on http enabled website.



    4. UNIT-III

    1. Write a program to sign and verify a document using DSA algorithm.

    2. Develop a Mobile application to create a notification in Android.

    3. Create animations and graphical primitives in Android environment.
